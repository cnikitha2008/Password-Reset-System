# Password Reset System

## Description
This project is a Password Reset System developed using Python, Pandas and NumPy.

## Features
- Login using Username or Email
- Forgot Password
- Random OTP Generation
- OTP Verification (3 Attempts)
- Password Validation
- Excel File Database

## Technologies Used
- Python
- Pandas
- NumPy
- Excel

## Project Files
- password_reset_system.py
- mails.xlsx
